# Rust Data Engineering Sandbox

This repository contains all projects and exercises from the Rust Data Engineering course.

## Structure

- module1/
- module2/
- module3/
- module4/

## Automation

Each module includes:
- formatting (cargo fmt)
- linting (cargo clippy)
- testing (cargo test)
- build (cargo build)

## Goal

Provide a reproducible, collaborative environment for learning Rust.
